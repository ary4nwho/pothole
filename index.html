Overview

The Pothole Detection and Alert System is a smart solution designed to identify potholes on roads using sensors and GPS data. The collected data can be used by municipal authorities for quick maintenance, helping prevent accidents and improve road quality.

Features

  • Automatic pothole detection using sensors.
  • Real-time location tracking with GPS.
  • Data storage for analysis and mapping.
  • Alert system for drivers and authorities.

How It Works

The system uses an ESP32 microcontroller connected with sensors like the MPU6050 accelerometer to detect road vibrations. Once a pothole is detected, the location is captured via GPS and can be displayed on a map or stored for later reporting.

Key Components

  • ESP32 Microcontroller
  • MPU6050 Accelerometer Sensor
  • GPS Module
  • SIM Module for Communication
  • Power Supply
